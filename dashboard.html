Plain-English Explanation · For Resort Managers

How This Dashboard Works
Written for a Karma resort manager who has never written code
1

What the dashboard shows: This screen displays key facts about each Karma member — which resort they prefer, how many times they've visited, how much they've spent, and how happy they are (out of 100%). The team who would use it daily: Guest Relations, Member Services, and Resort Managers who want to know at a glance which members deserve a personal touch.

2

What the "Generate AI Insight" button does: When you click it, the dashboard silently sends that member's information — their name, tier, visits, and satisfaction score — to an AI assistant running on this same computer. Within a few seconds, the AI reads all that data and writes back a short, personalised recommendation: for example, "Offer Marcus a complimentary yacht excursion — his 14 visits and 97% satisfaction make him an ideal ambassador." It works like sending a WhatsApp message to a very knowledgeable colleague, except the colleague replies instantly.

3

Why the AI runs on the laptop, not the internet: The AI model (called Llama 3.2) is installed and runs entirely on this computer — it never sends member data to Google, OpenAI, or any outside company. This means member names, spending habits, and personal details stay completely private inside Karma's own systems, satisfying data-privacy obligations without any cloud subscription costs.

4

How it was built: The entire dashboard is a single file that opens directly in a web browser — no app installation, no server, no IT setup required. Think of it like opening a PDF: double-click, it works.

5

What would need to change to connect real Karma member data: Right now the member records are sample data typed directly into the file. To connect to live Karma data, a developer would replace that sample data with a secure link to Karma's existing membership database (CRM). The dashboard design, the chart, the AI button — everything else stays exactly the same. It's a straightforward one-step upgrade.
